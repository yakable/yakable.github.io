Github Pages - Subpages

Das hat jetzt zwar etwas länger gedauert als erwartet, aber ich konnte mich doch dazu bewegen einen Blogeintrag zu erstellen.

Heute geht es in aller Kürze um Unterseiten auf Github Pages. Ich muss zugeben, dass ich auf der Github Seite nicht fündig geworden bin, wie man eine einfache Unterseite erstellt. Das sollte doch nicht allzu schwer sein in der Einführung zu erwähnen.

Eine schnelle Suche auf Stackoverflow hat aber Abhilfe schaffen können. Eine Subpage wird gemäß der Ordner- und Dateistruktur des git Repositories erstellt.

Das heißt: Die Datei '0.html' die den Inhalt dieser Seite enthält liegt im Verzeichnis yakable.github.io/blogpost/

Um auf diese Seite zu gelangen genügt es an die folgende Adresse in eurem Browser zu navigieren: https://yakable.github.io/blogpost/0.html

Relativ unspektakulär aber bestechend einfach!
